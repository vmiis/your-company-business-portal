What is a module?

A module is a file that contains Html, JavaScript and CSS. The contents may like the following.


    <div>
    ...
    </div>
    <script>
    ...
    </sciipt>
    <style>
    ...
    </style>
</div>

It could be just a static page for showing information or it could be a data collection form. It can be hosted on any static web server and be loaded into a content container when needed. Click here to see an example of a very simple module (a simple home page), and Click here to see the source code on CodePen. A module should starts with tag and ends with </div>. In side the div, you can put any html elements, script and style for this module. A module can also be a complete web page (with html, head and body) that will be loaded into a shadow-dom or an iFrame. A module should be a bit of independent so that it can be responsible and maintained by a individual person.
