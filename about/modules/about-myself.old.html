About myself

My goal
    One of my goals is to create an open source modular distributed cloud business operating system with the following features.
    • All modules should be open source, reusable and should be able to be hosted on any static web server.
    • Business data and files should be stored in business owner's private database and storage.
    • No server side coding for business logic. All business logic should be implemented on client side with javascript
