Bookkeeping for Australia small business

Live application

This is a very basic bookkeeping. We record all transactions in a simple way. At end we can get various kinds of aggregation reports. We believe that if each transaction can be classified clearly we will get most kind of reports. So in order to achieve this we predefined all transaction items. The daily bookkeeping data entry work is just selecting an item (with auto suggestion), enter date and amount.

3 database tables are used in this application

20007172, transactions table id
20005380, predefined transaction item table id
20010377, business intelligence, monthly aggregation table id

If you fork or clone this application for your own business, you need to replace these 3 table id in main.json with yours.
