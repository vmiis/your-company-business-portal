How to add a module to collect contact information?

Live application preview

Steps:
  • Create a data collection module. Click here to see the code of the module
  • Add the url of the module into the index.module.json to tell the system where the module is.
    Click here to see the contents of the index.module.json
  • Add a link in the menu on the navigation bar, so that the module will be loaded in when user click this link. <a onclick="$vm.nav_load_module('panel_main_contact-us')">Contact us</a>
Module
  • html: use HTML 5 standard form and inputs as well as HTML 5 validation.
  • script: serialize form data and send the data to a server. The server side code may send the data to an email box or save the data to a database. In this demo application we send the data to a database server hosted by Wappsystem.
  • style: use bootstrap 4 so make the form responsive.
