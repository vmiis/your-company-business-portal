Module

Live demo
A module is a file that contains HTML div with 3 parts, HTML, JavaScript and Style. It will be loaded dynamically into the content div of the layout. It can be stored or hosted on any static server. The module's div ID must be set as D__@@ and the function name in the script must be F__@@. The _@@ will be replaced automatically with a unique number. The function F__@@ will run after the module is appended to the content div.
Let's start with a very simple "Hello World" module.
<div id=D__@@> <div> Hello World!<br> I am a module.<br> I can be hosted at any static web server.<br> I can be lorded into your single page application when you need. </div> <script> function F__@@(){ } </script> <style> #D__@@{ height:100%; color:#fff; background-color:#17293e; padding:20px; } </style> </div> Try it yourself
