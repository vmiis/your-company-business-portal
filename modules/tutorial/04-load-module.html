Load module into the content div

Live demo

Using pure JavaScript to build a single page application is not a simple job. Fortunately there are some JavaScript library can be used to make such application a lot easier. VM library is a light JavaScript library we will use to build our single page application.

The minimal code for loading and initial the VM library is as following.

<script> //------------------------------------ var app_init=function(){ $vm={};$vm.vm={}; $vm.version=1; $vm.reload=''; $vm.hosting_path=window.location.href.substring(0, window.location.href.lastIndexOf('/')); } app_init(); //------------------------------------ </script> <script src="https://framework.vmiis.com/distribution/vmframework.min.js"></script>

In order to load modules into the system, we need to build a module list so that the system will know where the modules are located. The code to build a module list is as follow.

$vm.module_list={ "hello-world":{"url":"https://codepen.io/vmiis/pen/JBrdMq.html"}, "second-module":{"url":"https://codepen.io/vmiis/pen/OwxNdg.html"} }

We also need to build link buttons for user to load modules. The link button html is as follow.

<a href="hello-world">Hello World</a> <a href="second-module">Second Module</a>

The code to load the module is as follow.

$('#header a').on('click',function(){ event.preventDefault(); var href=$(this).attr('href'); if(href!=undefined){ $vm.load_module_v2(href,'content',{}); } })

At last we put all code together and play it on codepen website.
