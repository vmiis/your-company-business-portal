Make a proper header, footer and background with CSS

Live demo

We have built a simple website framework for loading and displaying modules. Now we want to make the website looks better.

Add a background image.

body{ background-image: url(https://www.vmiis.com/sys/layout.jpg); background-size: cover; background-repeat: repeat-x; background-position: center top; background-attachment: fixed; }

Style the header and footer.

#header{ height:100px; background: rgba(0,0,0,0.5); } #vm_logo{ display:inline-block; padding-top:35px; padding-left:20px; cursor:pointer; float:left; } #vm_logo_label_A{ font-family: Arial, sans-serif; color:#fff; font-size: 280%; line-height:10px; display:inline-block; padding-top:10px; } #footer{ height:20px; color: #fff; padding: 3px; background-color: #333; font:80%/18px arial; text-align: center; }

Style the menu.

#vm_menu { margin-top:50px; margin-left:160px; overflow: hidden; background: rgba(0,0,0,0.0); display:inline-block; float:left; } #vm_menu > a { float: left; display: block; color: #f2f2f2; text-align: center; padding: 10px 8px; text-decoration: none; font-size: 14px; cursor:pointer; } #vm_menu > a:hover{ background-color: #555; color: white; border-radius:4px; }
