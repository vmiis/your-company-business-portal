A module is a file that contains a HTML div with 3 parts, HTML, JavaScript and Style. It will be loaded dynamically into the content div of the layout. It can be stored or hosted on any static server. The module's div ID must be set as D__ID and the function name in the script must be F__ID. The _ID will be replaced automatically with a unique number. The function F__ID will run after the module is appended to the content div. Let's start with a very simple "Hello World" module.
Hello World!
I am a module.
I can be hosted at any static web server.
I can be lorded into your single page application when you need.
