Basic principles

The following list is my basic principles when developing web applications based on my many years lessons.

  • Strong security is ALWAYS the top important
  • Server side coding: as less as possible or NOT AT ALL for any applications
    (sever side only provide data services - request/response services)
  • Client side coding: as much as possible or MUST ALL for any application
  • Client site pages MUST be possible hosted on any static server
  • Client site pages MUST be encapsulated
I follow the principles because I want to achieve the following goals

  • Fast development
  • Lowest maintenance
  • Flexible changing/update
