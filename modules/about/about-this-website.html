I build this website for demonstrating the following 3 concepts

1. Modular distributed web application
Building web applications with shared reusable modules distributed on different static web servers around the world and maintained by different people with different skills (jQuery,ReactJS,AngulaJS etc). Is this possible to help business people to handle a growing amount of work?

2. A Network linking business's data together like 'Facebook' linking people's social media data
Facebook services let people be linked together (linking unstructured data). Is there any possibility to help business owners to link their business together for collaboration (linking structured data)?

3. A SaaS (Software as a service) network
Applications in Google Play have no data sharing functions. Can we build a SaaS network where the apps have a common data sharing interface?


This website is build with the following key features:

  • Single page application –– fast response, batter user experience
  • Static web hosting from anywhere –– fast, secure and free or low cost (this website is hosted on GitHub for free)
  • No server side coding –– simple development and easy maintenance
  • Modular distributed –– scalable, flexible, reusable and easy replaceable
  • Micro services for data –– directly read/write data from/to a database/files server with APIs
  • Flexible data access –– access is fully controlled by data owner
